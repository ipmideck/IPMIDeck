# IPMIDeck Documentation

The full documentation lives at **[docs.ipmideck.com](https://docs.ipmideck.com)** — start with
[Getting Started](https://docs.ipmideck.com/en/getting-started).

The docs are built with Mintlify and maintained in a separate repository; they are deliberately
**not** mirrored here, so there is a single source of truth.

This folder only holds static assets (e.g. [`screenshots/`](screenshots/)) referenced by the root
[README](../README.md).
